Using ucxdsa
============

This page describes how to use the ucxdsa package.

Installation
------------

.. code-block:: bash

    pip install ucxdsa

Quick Start
-----------

.. code-block:: python

    from dsa import array

    a = array.Array([1, 2, 3])
    a.append(4)
    print(a.to_list())

Modules
-------

The package contains modules for:

- array
- stack
- queue
- linked lists
- graphs
- and more

Refer to the documentation for details on each module.
